# Java类加载器ClassLoader

java.lang.ClassLoader抽象类
java.security.SecureClassLoader实现类 extends ClassLoader
java.net.URLClassLoader extends SecureClassLoader
sun.misc.Launcher.AppClassLoader extends URLClassLoader
sun.misc.Launcher.ExtClassLoader extends URLClassLoader

# API

# loadClass(String)、loadClass(String, boolean resolve)

通过二进制名来加载类，方法的默认实现查找类的顺序如下

  • 调用findLoadedClass检查类是否已经加载
  • 调用父类加载器的loadClass方法，如果父类为null,则使用jvm内置的类加载器
  • 调用findClass(String)方法查找类 resolve如果为true，类加载器则链接类。
