Once you write Java, you will write once.

# 从经典的HelloWorld程序说起

可选：如果你的机器已安装IDEA和Git，可使用git克隆项目到本地，并用IDEA打开项目，但如果你是初学者，建议还是按照如下步骤走一遍 如果暂时不想折腾其他的，只想简单了解下语言，那么完全可以使用记事本，notepad++等编辑器。当然你必须安装了jdk。

git clone https://github.com/thinkbuildshare/learnjavabydo.git

# 1. 创建HelloWorld.java

在D盘新建文件夹learnbydo，在learnbydo文件夹下新建HelloWorld.java，输入如下代码

注意：如果在Windows用记事本新建HelloWorld.java文件，且新建后的文件的图标并没有发生变化，那么需要检查文件夹选项->查看->隐藏文件和文件夹->显示隐藏的文件、文件夹和驱动器是否勾选。

# 4. 分析HelloWorld.java

已入行的程序员可能在学习不同语言时，都会遇到HelloWorld程序。甚至认为它简单得可笑，确实，HelloWorld足够简单，但毕竟是完整的程序， 其作为学习一门语言的切入点，简直不能再经典了。下面我们来挖挖HelloWorld中包含的知识点。

寥寥几行代码，对于学过其他语言的人来说，这不难理解，对于没有任何语言经验的入门者，我们这边对照着代码来简单梳理下。

// HelloWorld类
public class HelloWorld {

    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

  • 注释

    • 第一行// HelloWorld类是注释，不参与编译，被编译器忽略。
    • 在Java中使用//来注释一行。使用/* */来注释单行或多行。用/** */书写文档注释。
    • 注释是写给其他程序员了解代码意图或描述API参数和功能的。个人建议注释少而精，对于变量、方法等命名，需要见名知义，而不是靠注释来额外说明， 另外不建议随意写注释的原因是注释的维护成本高，在修改代码的同时，还需要修改注释，否则注释是无效的，甚至有误导的作用，在那种情况下，没有注释比错误的注释好。
    • 多行注释是不支持嵌套写法的。如/*外层注释 /*内层注释*/ */这样的写法是错误的，外层注释的开始/*将匹配内层注释的结束*/。引起编译错误
    • 注释的一个额外作用是临时修改代码，比如调试时临时注释掉代码行来查看运行效果。

  • 标识符、关键字、保留字

    • 标识符是程序中用来命名包、类、字段、方法、变量等元素的符号。如程序中的HelloWorld，main，args。
      • 标识符不能以数字开头。
      • 标识符是大小写敏感的。
    • 关键字是语言中用作特殊用途的符号。如public，class，static，void等。保留字和关键字都不得被用作自定义标识符。在IDE中通常关键字和保留字会显示为特定的颜色。
    • 保留字是语言预占的一部分符号。
关键字 用途或含义简介
public 公开级访问修饰符
class 定义类
static 静态修饰符
void 返回类型，表示无返回值

  • 定义类

    • Java是面向对象的语言，类是创建对象的模板。因此，我们编写Java程序一个重要的步骤是编写类。定义一个类很简单，Java中用class 来定义类。如下
      class 类名 { ... }
    • Java类名
      • 我们从标识符的约束中知道不能以数字开头，且不能使用保留字和关键字，除此之外，我们尽量以能表达意图和功能的名称。
      • Java中采用Pascal命名，即大驼峰为类命名。如HelloWorld，每个单词的首字母采用大写。
    • 如果使用public来修饰类的定义，那么需要满足这个类名必须和文件名一致。（包括大小写）
    • Java文件中最多只能有一个public修饰的类
    • Java文件中可以有多个类。

  • 定义方法

    • 方法由方法声明（修饰符、返回类型、方法名、参数）和方法体构成
      修饰符 返回类型 方法称 (参数类型 参数名, ...) {
   //这边是方法体
}
    • 方法是由名称和方法参数共同区分的，返回类型和修饰符不参与区分，如下返回类型不同的两个方法不能定义在同一个类中，编译期会提示a方法已存在
      int a() {
  return 0;
}

double a() {
  return 0d;
}
    • 具有不同参数个数或者不同参数类型的函数可以具有相同的函数名称，此时我们称为重载方法
      int a(int a) {
  return 0;
}

int a(String a) {
  return 0;
}

  • main方法

    • Java应用程序的执行入口，即程序将从main方法开始执行。
    • 方法声明必须是public static void main(String[] args)，其中args可以是任意合法标识符，public和static的顺序可以对换
    • 参数args是字符串数组类型，在执行java命令运行程序时，可以作为命令行参数传入，我们的程序没有演示，感兴趣的朋友可以试下，如下
          java HelloWorld "I am comming" "第二个参数"
      可能你会疑问这个参数干嘛用呢？我们可以用这些参数控制程序的行为。类似于一种配置，通过在外部传入参数，达到控制程序行为而无需改动代码。 是不是有点像遥控器，我按什么按钮，电视识别并做对应的响应。

  • 打印HelloWorld
    System.out.println("Hello World");

    • 使用了核心库中的System类
    • 调用System类的公共静态字段out，继续调用了out对象的println方法，最终在控制台打印出Hello World
